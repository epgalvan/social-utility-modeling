Social Utility Modeling Tutorials

About these Tutorials

These tutorials are completed data analyses on existing, open access datasets which you can download yourself. These tutorials complement the handbook in the sense that they demonstrate how the code provided in the handbook can be correctly implemented, making it valuable for passive viewing and checking your work! You can learn more about these datasets and how to download them here. You can also download the notebooks themselves here.

See the Paper

Alongside these tutorials and the handbook, we also wrote a companion paper which is currently in review at Social Cognition. You can read the preprint for this paper here.

About the Handbook

This webpage serves as a resource to guide you through computational modeling of social utility, but it is not the handbook itself. The handbook is an open-source, freely available project on Read the Docs which provides both lessons and these tutorials: you can complete the tutorials piece by piece as you learn from the lessons.
